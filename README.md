# Raven-Programmer
An implementation of a 68HC705E1 Programmer, to replace the 68HC05E1 Mask ROM Microcontroller that was used as the EGRET and CUDA ICs in 90s Macintoshes, with the EPROM programmable 68HC705E1 part.
